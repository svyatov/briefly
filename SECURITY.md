# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✅        |

## Reporting a vulnerability

Please report privately, not in a public issue:

- [GitHub Security Advisories](https://github.com/svyatov/briefly/security/advisories/new), or
- email `leonid@svyatov.com`

You'll get an acknowledgement within 48 hours and an assessment within a week. Fixes are disclosed
coordinately once a patched release is available.
